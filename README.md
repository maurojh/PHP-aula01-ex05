"# PHP-aula01-ex05" 

A função strpos() busca um texto em uma string e retorna a posição da primeira ocorrência da palavra. Se não for encontrada ela retorna false.

Sabendo que o comando IF testa uma condição e executa um comando caso a condição seja diferente de FALSE. Escreva a página procura.php que recebe o texto e a palavra procurada e verifica se existe no texto a palavra. A palavra e o texto são enviados pela página ex3.php.
